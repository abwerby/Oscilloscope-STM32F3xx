# Oscilloscope

Oscilloscope based on stm32f303re and ili9486 3.5-inch TFT LCD screen

## Major features 

1- Analog bandwidth: 0 – 400KHz  
2- Sampling rate: 2Msps max  
3- Record length: 4096 points  
4- Timebase: 1us/Div – 100us/Div  
5- Sensitivity: 3.23mV
